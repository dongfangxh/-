 # xss

##Cross Site Script

####中文名称：跨站脚本攻击

####危害：盗取用户信息、钓鱼、制造蠕虫等

## 危害：

####通过**HTML注入**篡改网页，插入了恶意脚本，当用户在浏览网页时。实现控制用户浏览器行为的一种攻击方式。
